# gp-dashboard-project# London GP Access Inequality Dashboard

This project is an interactive dashboard that explores how deprivation affects access to GP services across London boroughs. It combines GP access data, patient experience data, and deprivation scores to highlight inequalities in primary care.

The aim is to make complex health data easier to understand for non-technical users such as policymakers and local authorities.

## Features

- Borough filtering (interactive sidebar)
- Bar charts comparing GP access and patient experience
- Scatter plot showing relationship between deprivation and GP access
- Simple and user-friendly interface
- Data visualisation using Plotly

---

## Technologies Used

- Python
- Pandas
- Plotly
- Streamlit
